# 🧮 Simple Calculator (C# WinForms)

## 📖 Description
This project is a simple calculator application built using C# Windows Forms. It performs basic arithmetic operations such as addition, subtraction, multiplication, and division.  

The goal of this project is to demonstrate understanding of event-driven programming, user input handling, and basic logic implementation in C#.

---

## ⚙️ Features
- Basic arithmetic operations (+, −, ×, ÷)  
- Continuous calculations (chaining operations)  
- Clear/reset functionality  
- Error handling for division by zero  
- Simple and responsive GUI using Windows Forms  

---

## 🛠️ Technologies Used
- C# (.NET Windows Forms)  
- Visual Studio  

---

## 🚀 How to Run
1. Open the project in Visual Studio  
2. Build the solution  
3. Run the application  
4. Use the on-screen buttons to perform calculations  

---

## 🧠 What I Learned
Through this project, I learned how to handle button events in Windows Forms, manage program state using variables, and implement basic arithmetic logic. I also gained experience in structuring user input and controlling application flow in an event-driven environment.

---

## 💡 Notes
- Supports single-step and chained calculations  
- Designed for basic desktop use and learning purposes
